# Polices Surplasse (auto-hébergées)

Quatre familles, toutes sous licence SIL Open Font License 1.1 (OFL), redistribuables :

- **Archivo** (Omnibus-Type) : police principale, titres et UI. Poids 400 a 900 (variable).
- **Space Mono** (Colophon Foundry pour Google) : la meta (prix, quantites, horodatages). Poids 400 et 700.
- **Bodoni Moda** (Owen Earl) : mot-symbole Surplasse. Poids 400, axe optique fixe a 72 pour une signature stable.
- **Parisienne** (Astigmatic) : script, reserve aux noms d'etablissements et aux accents editoriaux. Poids 400.

Fichiers `woff2` recuperes depuis Google Fonts (sous-ensembles latin, latin-ext, vietnamese), servis en local pour la performance et la confidentialite (pas de CDN tiers). Chaque famille a son fichier CSS `@font-face` deja localise. La licence de Bodoni Moda est conservee dans `bodoni-moda-OFL.txt`.
